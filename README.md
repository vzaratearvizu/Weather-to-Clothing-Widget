# Weather Clothes

A Windows desktop weather widget that shows hourly forecasts for Boston and recommends what to wear based on the feels-like temperature, precipitation, and wind conditions. Especially useful for not having to guesstimate what to wear or bring when away from home for a majority of the day.

## Features
- Hourly temperature, feels-like, wind, gusts, and precipitation
- Outfit recommendations per hour (jacket, hoodie, umbrella, etc.)
- Color-coded background per weather condition
- Infinite scroll through today and tomorrow's forecast
- Auto-refreshes every hour

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`

## How to Build
Run `npm run build` — this generates a standalone `weather-widget.exe` in the `dist/` folder.
